# 质量检查详细指引

本文档为 SKILL.md 中质量检查清单的详细说明，提供判断标准、常见错误和修正方法。

## 1. 是否准确对应当前文章

**判断标准**：摘译中的核心论点、关键事实、行为体、时间背景必须与原文一致。

**常见错误**：
- 套用相似主题的模板（如把"欧盟对俄制裁"的摘译套用到"欧盟对华政策"）
- 混淆不同文章的作者观点
- 把其他文章的案例或数据写入当前摘译

**修正方法**：
- 重新核对原文标题、作者、发布时间
- 逐句检查摘译中的事实是否出现在原文中
- 删除所有不在原文中的信息

## 2. 是否覆盖全文中心思想、主要论证结构和政策含义

**判断标准**：
- 中心思想：作者最想表达的核心判断或主张
- 论证结构：支撑核心判断的主要逻辑链条（通常 2-4 条）
- 政策含义：作者明确提出的政策建议或规范性判断

**常见错误**：
- 只概括背景和现象，没有提炼作者的核心判断
- 罗列大量细节，但没有呈现论证逻辑
- 遗漏作者的政策建议部分

**修正方法**：
- 问自己：作者写这篇文章是为了论证什么？
- 检查是否包含"作者认为""报告指出""基于此，作者建议"等归因表达
- 如果字数不够，优先保留核心判断和政策建议，压缩背景和案例

## 3. 是否包含作者最重要的政策建议

**判断标准**：原文结论部分、政策建议部分、或带有"应该""需要""建议"等规范性表达的内容。

**常见错误**：
- 只写"作者提出了若干建议"，但不说明具体是什么
- 把描述性判断（"X 正在发生"）当作政策建议
- 遗漏作者最强调的建议（通常在结论第一段或单独标题下）

**修正方法**：
- 重新阅读原文结论部分
- 使用"报告建议……""作者主张……""因此，……应……"等表达
- 如果有多条建议，选择作者最强调的 1-2 条

## 4. 是否没有遗漏关键行为体及其关系

**关键行为体定义**：对文章核心论点有决定性影响的国家、组织、机构或个人。

**判断标准**：
- 如果原文讨论"美国对华政策"，关键行为体是美国和中国
- 如果原文讨论"欧盟内部分歧"，关键行为体是欧盟及其主要成员国
- 如果原文讨论"WTO 改革"，关键行为体是 WTO、主要成员国、改革提议方

**常见错误**：
- 只写"某国"而不明确是哪个国家
- 遗漏对立方或受影响方（如只写制裁方，不写被制裁方）
- 把次要行为体当作主要行为体（如把某智库当作政策主体）

**修正方法**：
- 列出原文中所有国家、组织、机构名称
- 判断哪些是核心论点的主语或宾语
- 确保摘译中明确写出这些行为体及其关系（合作、对抗、依赖等）

## 5. 是否没有把作者观点写成自己的延伸判断

**判断标准**：摘译中的所有判断都应该来自原文，使用归因表达。

**常见错误**：
- 把"作者认为 X 可能导致 Y"写成"X 将导致 Y"（去掉了不确定性）
- 把"作者担忧 X"写成"X 是一个严重问题"（把担忧写成事实）
- 添加原文没有的因果关系或推论

**修正方法**：
- 检查每个判断句，确认是否有"作者认为""报告指出"等归因
- 保留原文的不确定性表达（"可能""担忧""风险"）
- 删除所有"显然""毫无疑问""必然"等自己添加的强化表达

## 6. 是否没有添加原文没有的内容

**原文没有的内容包括**：
- 实时新闻（如"最近 X 国发生了……"，但原文发布时这件事还没发生）
- 外部背景（如"众所周知，X 国一直……"，但原文没有提及）
- 个人评价（如"这一政策非常重要"，但原文只是客观分析）
- 其他文章的观点或数据

**常见错误**：
- 为了让摘译更完整，补充了原文没有的背景
- 把自己对该议题的了解写入摘译
- 用最新新闻"更新"原文的过时信息

**修正方法**：
- 逐句对照原文，确认每个信息点都有出处
- 删除所有"众所周知""近年来""一直以来"等可能引入外部信息的表达
- 如果原文信息不完整，保持不完整，不自行补充

## 7. 是否没有重复表达、口号化表达或聊天式尾巴

**重复表达**：
- 在不同段落中用不同措辞表达同一个观点
- 背景部分和论证部分重复同一事实

**口号化表达**：
- "具有重要意义""深远影响""战略高度"等无具体指向的套话
- "日益复杂""不断变化"等空泛描述

**聊天式尾巴**：
- "以上是对该文章的摘译"
- "如需修改请告知"
- "希望对您有帮助"

**修正方法**：
- 删除重复段落，保留表达最清晰的一处
- 用具体机制替代套话（如"影响欧盟能源安全"而非"具有重要影响"）
- 删除所有正文之后的聊天式表达，确保以"范钶烨 摘译"结尾

## 8. 是否控制在用户要求字数内

**字数统计方法**：统计所有中文字符、英文单词、数字，不包括标点符号和空格。

**常见超标原因**：
- 保留了过多案例细节
- 背景部分过长
- 重复表达

**压缩优先级**（从高到低）：
1. 删除次要案例和数据
2. 压缩背景部分，只保留理解核心论点必需的背景
3. 合并重复表达
4. 简化过渡句和连接词
5. 最后才考虑压缩核心论点和政策建议

**常见不足原因**：
- 只概括了部分内容
- 遗漏了论证结构或政策建议

**扩充优先级**（从高到低）：
1. 补充遗漏的政策建议
2. 补充主要论证逻辑
3. 补充关键事实和数据
4. 补充必要背景
5. 不要为了凑字数而添加空泛表达

## 9. 是否使用中文标题和中文时间

**中文标题**：
- 不保留英文标题，除非专名确需保留（如"CBDC""WTO"等缩写）
- 标题应自然中文化，不机械直译

**中文时间格式**：
- "2026年4月"（有年月）
- "2025年1月15日"（有具体日期）
- 不使用"April 2026""2026-04-15"等英文或数字格式

## 10. 是否以"范钶烨 摘译"结尾

**检查要点**：
- 署名必须紧接正文最后一句，中间只有一个空格或标点
- 署名只出现一次，不在开头或中间出现
- 署名之后没有其他内容（如链接、说明、聊天式尾巴）

**正确示例**：
```
……因此，欧盟需要在经济利益与安全关切之间取得平衡。范钶烨 摘译
```

**错误示例**：
```
……因此，欧盟需要在经济利益与安全关切之间取得平衡。

范钶烨 摘译

以上是对该文章的摘译。
```
